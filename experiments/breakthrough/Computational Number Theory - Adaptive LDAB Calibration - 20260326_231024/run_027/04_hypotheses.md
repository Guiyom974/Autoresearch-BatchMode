#### Quick checklist for the experimental workflow

1. **Fix overflow** – switch to big‑integer accumulators (H1).  
2. **Fix memory truncation** – replace fixed‑size buffers with dynamic/streaming storage (H5).  
3. **Run both truncated and full pipelines** for k = 3‑7 to reproduce the 1.17 artifact (H2) and confirm the corrected trend.  
4. **Compute exact \(R(8),R(9)\)** using the fully‑corrected pipeline (H3).  
5. **Fit models and compare AIC/BIC** (H4).  

If all five experiments return the predicted outcomes, the research problem is solved: the pipeline is clean, the exact high‑order ratios are known, and the true asymptotic scaling law is identified. If any hypothesis fails, the failure pin‑points exactly which component of the pipeline still needs debugging.
