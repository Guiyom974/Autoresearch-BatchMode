### 1. Universal-Primorial Prime Construction (UP 2 Method)
A recent paper introduces the "Universal-Primorial Prime Construction" (UP 2 Method), which is a deterministic framework for generating and testing extremely large prime candidates. It uses an exponential primorial base combined with a universally defined integer seed derived from mathematical and physical constants. The method embeds a wheel sieve directly into the exponential growth mechanism, eliminating divisibility by all primes up to a prescribed cutoff while preserving computational tractability at record-scale digit lengths.

### 2. Primorial Numeral System (OEIS Wiki)
The primorial numeral system (or primoradic) is a mixed-radix numeral system where the place values are successive primorial numbers ($p_i\#$). The $i$-th primorial is the product of the first $i$ primes. In this base, the maximum value allowed in any digit position varies according to the next prime. The sequence of integers written in primorial base is cataloged in the OEIS as **A049345**. For larger numbers, digit separators (like colons) are used because the "digits" can exceed 9 (e.g., the radixes are 2, 3, 5, 7, 11, etc.).
**Source:** [OEIS Wiki - Primorial numeral system](https://oeis.org/wiki/Primorial_numeral_system) [[2]](https://oeis.org/wiki/Primorial_numeral_system)

### 4. Fractions and Terminating Digits in Primorial Base
Mathematician Michael De Vlieger has contributed extensive notes on how fractions and digit sequences behave in primorial bases compared to standard bases. Because a primorial base has only one instance of a given prime in its radix sequence, fractions behave differently. For instance, $1/4$ does not terminate in primorial base, but $1/6$ does (since 6 is squarefree). If a fraction has a multiplicity of any prime divisor in its denominator, it will result in a mixed-recurrent digit sequence (a non-repeating prefix followed by a repeating suffix), similar to how $1/6$ behaves in base 10.
**Source:** [Michael De Vlieger's OEIS Contributions](https://www.vincico.com/proof/oeis.html) [[4]](https://www.vincico.com/proof/oeis.html), [[4]](https://www.vincico.com/proof/oeis.html)

### 5. Weird Number Bases (DataGenetics)
This article explores mixed radix systems, specifically highlighting the "Primorial Base". It explains that unlike traditional positional notation where moving left increases the exponent of a static base, primorial bases use columns representing primorials (1, 2, 6, 30, 210, etc.). Because primorials are square-free integers and each contains more distinct prime factors than any smaller number, representing numbers in this base creates unique digit sequences where the maximum allowed digit in the $n$-th position is $p_{n+1} - 1$.
**Source:** [DataGenetics - Weird Number Bases](http://datagenetics.com/blog/december22015/index.html) [[5]](http://datagenetics.com/blog/december22015/index.html)
